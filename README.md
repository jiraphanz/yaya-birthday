หน้าเว็บนี้ออกแบบสำหรับหน้าโทรศัพท์เพื่อใช้เป็นหน้า Surprise วันเกิด

ไฟล์สำคัญ:
- index.html
- styles.css
- script.js

วิธีทดสอบท้องถิ่น:
1. เปิดไฟล์ `index.html` ในเบราว์เซอร์ (หรือรัน Live Server ใน VS Code)

วิธีโฮสต์บน GitHub Pages:
1. สร้าง repository บน GitHub และ push โค้ดทั้งหมด
2. ในหน้า repo ไปที่ Settings → Pages
3. เลือก branch `main`/`master` และ folder `/ (root)` แล้วกด Save
4. รอไม่กี่นาที แล้วหน้าเว็บจะอยู่ที่ `https://<your-username>.github.io/<repo-name>/`

ต้องการให้ผมช่วย commit และ push ขึ้น GitHub หรือปรับแต่งข้อความ/ดีไซน์เพิ่มเติมไหม?
